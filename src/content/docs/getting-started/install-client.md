# 客户端安装教程

## Java 环境安装

1. 下载并安装对应版本的 Java
2. 安装后确认系统可正常识别
3. 避免多个版本冲突

## 整合包导入

1. 下载整合包
2. 在启动器中导入整合包
3. 等待资源和模组加载完成

## 首次启动检查

- 分配合适内存
- 确认模组完整
- 确认资源包正常
- 进入测试环境确认无报错

# 如何加入服务器

欢迎来到郑爱玩MC。

## 加入前准备

- 确认服务器支持的游戏版本
- 确认是否需要白名单
- 确认是否需要加入 QQ 群或 QQ 频道
- 确认是否需要使用整合包

## 入服流程

1. 安装正确版本的 Java
2. 下载客户端整合包
3. 导入启动器并完成第一次启动
4. 联系管理员或加入群聊完成审核
5. 使用服务器地址进入

## 遇到问题怎么办

如果你遇到启动失败、进服失败、资源包报错等问题，请先阅读常见问题页。
